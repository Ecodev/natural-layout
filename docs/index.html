Getting started

Available breakpoints prefixes : xs, sm, md, lg, xl.

Usage : class="[breakpoint-]nat-class[-size]"

Examples :

Greater than medium (included) change layout to horizontal

class="nat-vertical  md-nat-horizontal"

Greater than medium (included) increase gap

class="nat-gap  md-nat-gap-50"

SCSS usage : media query

@use '@ecodev/natural-layout' as nat;
#id {
    // apply max-width for sizes bigger than md (included)
    @include nat.gte(md) {
        max-width: 80%;
    }

    // apply color red for sm and md sizes
    @include nat.between(sm, lg) {
        color: red;
    }
}
